Este diretório contêm tutoriais gerados a partir de screenshots, eles foram gerados pelo programa Wink 2.0, disponível apenas para Windows em http://www.debugmode.com/wink/


Descrição dos arquivos:

diretorio_do_tutorial
  \
   |__ imagens                   --> Contêm as screenshots feitas previamente
   |      |__ imagem-00.xcf      --> Imagem que descreve o tutorial no formato do GIMP
   |      |__ imagem-#.png       --> Todos os screenshots feitos
   |__ tutorial.[htm, js, swf]   --> Arquivos de saída gerados pelo Wink
   |__ tutorial.wnk              --> Arquivo de projeto do Wink, que contém todo o conteúdo do tutorial e pode ser modificado usando o Wink.
